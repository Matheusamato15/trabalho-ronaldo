1- Do que se trata a especificacao caixa preta?

A caixa preta especifica o comportamento de um sistema ou de uma parte dele. O sistema (ou parte) responde a estimulos especificos (eventos) aplicando um conjunto de regras e transicao que mapeiam os estimulos em uma resposta.


2- Do que se trata a especificacao caixa de estado?

A caixa de estado encapsula dados de estado e serviços (operacoes) de maneira analoga a objetos. Nessa visao de especificacao, sao representadas entradas para a caixa de estado (estimulos) e saídas (respostas). A caixa estado representa o “historico de estimulo” da caixa preta; isto e, os dados encapsulados na caixa de estado que devem ser retidos entre as transicoes inferidas.


3- Do que se trata a especificacao caixa clara?

As funcoes de transicao inferidas pela caixa de estado sao definidas na caixa clara. Em outras palavras, uma caixa clara contem o projeto procedimental para a caixa de estados.


4- Qual e o objetivo de teste sala limpa?

O objetivo de teste sala limpa e validar requisitos de software demonstrando que uma amostragem estatistica de casos de uso foi executada de forma bem sucedida.


5- Cite uma definição dos metodos formais.

Metodos formais usados no desenvolvimento de sistemas de computadores sao tecnicas de base matematica para descrever propriedades de sistemas. Esses metodos formais proporcionam uma estrutura dentro da qual as pessoas podem especificar, desenvolver e verificar sistemas de maneira sistematica, em vez de casual.
